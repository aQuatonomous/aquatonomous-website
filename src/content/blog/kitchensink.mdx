## <a name="Code"></a>Code

### Inline code

Water-inspired design: `flow(text, typography)` reflects the smooth, unbroken motion of a current in code. Typography flows as naturally as streams converging into rivers.

```go
package main

import (
    "fmt"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Welcome to the ocean of design, %s!", r.URL.Path[1:])
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}
```
